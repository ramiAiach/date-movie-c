Projet Date & Movie – Programmation C

Objectif
--------
Ce projet a pour objectif de mettre en pratique la programmation modulaire en C
à l’aide de la compilation séparée.

Il comprend :
- Un module Date (création, validation, affichage)
- Un module Movie utilisant une structure Date
- Un programme principal servant de tests

Le projet est compilé à l’aide d’un Makefile et est compatible avec les systèmes Linux.

Compilation et exécution
------------------------
1. Compiler le projet :
   make

2. Exécuter le programme :
   ./test_date_movie

Technologies utilisées
----------------------
- Langage C (C99)
- GCC
- Make
- Linux / macOS
